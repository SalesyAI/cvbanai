# CVBanai Brand Guidelines

> Modern, professional Resume/CV Builder with a premium dark theme

## Brand Identity

**Company Name:** CVBanai
**Tagline:** "AI-Powered Resumes That Get You Hired"
**Target Audience:** Job seekers, professionals, career changers

## Color Palette

### Primary Colors
```css
--primary-500: #8b5cf6;     /* Violet - Main brand color */
--primary-600: #7c3aed;     /* Darker for hover states */
--primary-400: #a78bfa;     /* Lighter for accents */
```

### Secondary Colors
```css
--accent-500: #06b6d4;      /* Cyan - Secondary accent */
--accent-600: #0891b2;      /* Darker for hover */
--success-500: #10b981;     /* Green - Success states */
--warning-500: #f59e0b;     /* Amber - Pro badges */
```

### Neutral Colors (Dark Theme)
```css
--gray-950: #030712;        /* Darkest background */
--gray-900: #0f172a;        /* Main background */
--gray-800: #1e293b;        /* Card backgrounds */
--gray-700: #334155;        /* Borders, dividers */
--gray-500: #64748b;        /* Muted text */
--gray-300: #cbd5e1;        /* Secondary text */
--gray-100: #f1f5f9;        /* Primary text */
--white: #ffffff;           /* Headings, emphasis */
```

### Gradient Definitions
```css
--gradient-hero: linear-gradient(135deg, #8b5cf6 0%, #06b6d4 100%);
--gradient-card: linear-gradient(180deg, rgba(139,92,246,0.1) 0%, transparent 100%);
--gradient-glow: radial-gradient(circle, rgba(139,92,246,0.3) 0%, transparent 70%);
```

## Typography

### Font Families
- **Headings:** Inter (700, 800 weights)
- **Body:** Inter (400, 500, 600 weights)
- **Monospace:** JetBrains Mono (for resume preview)

## Visual Language

### Design Mode
- [x] Dark Mode (default)

### Animation Style
- **Intensity:** Smooth and professional
- **Transitions:** 300ms ease-out default
- **Loading states:** Pulsing dots, skeleton loaders
- **Hover effects:** Subtle scale (1.02) + glow

### Visual Elements
- **Glassmorphism:** Yes, for modals and cards
- **Gradients:** Violet to cyan accent gradient
- **Shadows:** Colored shadows (violet glow)
- **Borders:** 1px with low opacity white

## Voice & Tone

### Copy Style
- **Tone:** Confident, encouraging, professional
- **Language:** Clear, action-oriented
- **Personality:** Helpful career partner

### CTA Language
- Primary: "Start Building Your Resume", "Refine with AI"
- Secondary: "Upload Your Resume", "See the Magic"
- Upgrade: "Unlock Pro Features", "Get Hired Faster"
